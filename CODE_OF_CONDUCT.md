# Code of Conduct

## Our Standard

Inflexis is a professional engineering organization. All contributors and community members are expected to behave professionally in every interaction.

**We expect:**
- Respectful, constructive communication
- Technical feedback focused on the work, not the person
- Good faith in reporting bugs and requesting features
- Respect for proprietary boundaries (MAO Platform internals are not open source)

**We do not tolerate:**
- Harassment, personal attacks, or discriminatory language
- Publishing private or sensitive information about anyone
- Attempts to extract proprietary source code or trade secrets
- Spam, self-promotion, or off-topic solicitation

---

## Scope

This code of conduct applies to all Inflexis GitHub repositories, issue trackers, pull requests, and any public communication using Inflexis infrastructure or representing Inflexis.

---

## Enforcement

Violations may result in:
- Comment removal
- Issue or PR closure
- Temporary or permanent ban from Inflexis repositories
